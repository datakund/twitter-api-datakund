Installation/Usage:
*******************
You can find this package on Pypi (see `here <https://pypi.org/project/bot-studio/>`_).

Command to install :- ``pip install bot-studio``

Import datakund
**************************************************
.. code-block:: python

	from bot_studio import *

Creating Object
**************************************************
.. code-block:: python
	
	twitter=bot_studio.twitter()
	
It will return the object which you can further use to call twitter functions and opens a automated browser
